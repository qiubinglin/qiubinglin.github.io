# 介绍
资产负债表由两大区块构成，资产端的**资产合计**，负债端的**负债合计**与**所有者权益**，通常有
$$
    资产合计 = 负债合计 + 所有者权益
$$
资产合计可以细分为**流动资产**和**固定资产**。

# 不同部门的资产负债表
## 个人
资产端：储蓄、现金、房产、车等

负债端：房贷、信用贷款、自身所有者权益等

下表展示了简单的个人资产负债表：

| 项目      | 金额 |
| ----------- | ----------- |
| 储蓄      | 100       |
| 现金   | 100        |
| 房产   | 10000        |
| 车   | 1000        |
| ***总资产***   | 11200        |
| 房贷      | 9000       |
| 信用贷款   | 1000        |
| 所有者权益   | 1200        |
| ***总负债***   | 11200        |
